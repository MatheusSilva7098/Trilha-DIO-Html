Entendendo a diferença entre Listas ordenadas e não ordenadas

A principal diferença entre a lista ordenada e a lista não-ordenada está na maneira com que os itens de cada lista são apresentados - A lista ordenada enumera os elementos; a lista não-ordenada coloca marcadores antes de cada elemento
Sendo assim, temos uma compreensão mais do funcionamento das listas, aprendendo que o uso de cada lista varia de acordo com a necessidade do projeto

Entendendo HTML semântico

O HTML semântico é a de deixar o site com suas informações bem explicadas e compreensíveis para o computador, ajudando até mesmo em sua busca no Google e facilitando o entendimento de leitores de acessibilidade.
Assim sendo, o HTML semântico torna o código mais legível para todos, desde os desenvolvedores que irão trabalhar com ó código, até sistemas para web que irão realizar a leitura do mesmo.

Div e Span

A tag div é um elemento genérico de nível de bloco usado para associar e agrupar uma parte maior de uma página da web – geralmente uma seção, como um cabeçalho, rodapé, conteúdo principal e assim por diante.

A tag div deve ser utilizada quando agrupar grandes blocos de conteúdo e quando quiser dispor elementos na página da web.

A spantag é um elemento embutido que você usa para destacar uma parte menor do conteúdo com CSS ou JavaScript.

A tag span deve ser utilizada quando houver necessidade de modificar uma parte específica do conteúdo de forma independente.
